Este repositorio trata sobre un ejemplo práctico de carga, exploración, limpieza y visualización básica de un dataset.

La idea inicial era reestructurar los datos provinientes de los tres ficheros csv de la publicación. No obsante, para simplificar la tarea se ha decidido prescindir del fichero *steamspy_data.csv* para la entrega final.

Este readme es más bien corto porque la mayoría del proceso está escrito en los notebooks.
